# Omni-Chess ♟️

A chess analytics and player profiling platform built with Python, PostgreSQL, and machine learning.

## What it does
- Fetches any player's game history from the Lichess API
- Analyzes opening repertoire win/loss rates
- Profiles player style (aggressive, positional, time-trouble prone)
- Classifies blunder patterns using machine learning

## Tech Stack
- **Data**: Python, Pandas, NumPy
- **Database**: PostgreSQL
- **ML**: Scikit-learn, XGBoost
- **Backend**: FastAPI
- **Frontend**: React / Next.js (coming soon)
